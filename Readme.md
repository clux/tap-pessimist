# tap-pessimist
A small command line filtration tool for filtering some of the constant spam from node-tap.

## Usage
Pipe to:

````bash
$ producerApp | tap-pessimist
````

or

````bash
$ cat log.txt | tap-pessimist
````

## Installation

````
npm install tap-pessimist
````
